BlueWord — Production Site (Hosting-Ready)
============================================

업로드할 파일 목록 (Files to upload to your hosting server):

  index.html        ← 메인 페이지 (entry point)
  privacy.html      ← 개인정보처리방침
  terms.html        ← 이용약관
  assets/
    blueword-logo.png
    blueword-logo.svg

업로드 방법 (How to deploy):
---------------------------
1. 호스팅 서버의 public_html (또는 www, htdocs, public) 디렉터리에 위 파일들을
   동일한 폴더 구조로 업로드합니다. assets/ 폴더는 그대로 유지해 주세요.
2. 도메인 루트로 접속하면 index.html이 자동으로 표시됩니다.
3. 폰트(Pretendard, Archivo, Instrument Serif)는 Google Fonts / jsDelivr CDN에서
   자동 로드됩니다. 별도 설치 불필요.
4. 영웅 섹션 등의 이미지는 Unsplash CDN에서 로드됩니다.

호환성 (Compatibility):
---------------------------
- PC (Chrome / Edge / Firefox): ≥ 1100px 데스크톱 레이아웃
- Mac (Safari / Chrome): 동일
- 태블릿: 640–1100px 2-컬럼 자동 적응
- 모바일 (iOS Safari, Android Chrome): ≤ 640px 단일 컬럼 + 햄버거 메뉴
